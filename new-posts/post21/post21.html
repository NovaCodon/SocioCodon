Biggest change of Windows 11 26H2

30/06/2026 01:29 PM IST
Niral Bhatt

Windows 11 26H2 is bringing one of the most requested changes!

As per the latest Windows 11 26H2 Insider Developer Testing Preview (Experimental channel), Microsoft is testing a new option to disable Bing-powered web results in Windows Search, allowing Search to focus mainly on local files, apps, and settings.
