OpenCV Python Installation Demo for Class X, XI, XII and above

Published: 01/01/2026 1:54 PM IST
@NiralBhatt

First of all, Happy New Year 2026 to you all. Today we will learn about how to install the python library, OpenCV. It is a powerful library used for computer vision and artificial intelligence projects. Before starting any OpenCV-based work in Python, it is important to install it correctly on your system.

Step 1: Open Command Line

Open the command-line interface on your operating system:

  • Windows: Open Command Prompt
  • Linux / macOS: Open Terminal

Step 2: Install OpenCV

In the command line, type the following command and press Enter: 

pip install opencv-python

Wait for the installation to complete. Make sure your internet connection is active during this process.

Step 3: Verify the Installation

After the installation is finished, open Python IDLE (or any Python editor of your choice).

Type the following code: 

import cv2
print(cv2.__version__)

If OpenCV is installed correctly, this will display the installed OpenCV version number. This confirms that OpenCV has been installed successfully and is ready to use.

This method works well for students of Class X, XI, XII and anyone who wants to begin their journey with Python and Computer Vision.
