Easy Hand Trick to Remember Trigonometric Ratios

Published: 31/12/2025 6:03 PM IST · @NiralBhatt

Trigonometry can feel difficult when we try to memorize too many values. But there is a very simple hand trick that helps you remember all the trigonometric ratios of standard angles easily.

First, look at the fingers of your hand from the thumb to the little finger. Number them like this:

  • Thumb → 1
  • Index finger → 2
  • Middle finger → 3
  • Ring finger → 4
  • Little finger → 5

Now assign angles to each finger:

  • Thumb → 0°
  • Index finger → 30°
  • Middle finger → 45°
  • Ring finger → 60°
  • Little finger → 90°

To find the sine of any of these angles, use this formula:

sin(angle) = √(finger number / 4)

This gives you the sine values directly. To find other trigonometric ratios:

  • Cosine values are obtained by reversing the sine values
  • Tan = sine ÷ cosine
  • Cot = reverse of tan
  • Cosec = reciprocal of sine
  • Sec = reciprocal of cosine

Using this single hand trick, you can quickly remember all the trigonometric ratios for standard angles without memorizing long tables.

Trigonometric Ratios Table

Angle sin cos tan cot sec cosec
0° 0 1 0 ∞ 1 ∞
30° 1/2 √3/2 1/√3 √3 2/√3 2
45° 1/√2 1/√2 1 1 √2 √2
60° √3/2 1/2 √3 1/√3 2 2/√3
